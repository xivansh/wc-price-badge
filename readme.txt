Price Badge For WooCommerce : This plugin is used to mark prices with badge text.

Isi Form 
- Enable Badge > Centang untuk menampilkan Price Badge.
- Start Date & End Date > Menampilkan Price Badge sesuai tanggal yang ditentukan.
- Type Badge > Pilih untuk menggunakan badge text atau gambar.
- Color Badge > Untuk mengganti warna Price Badge sesuai keinginan.
- Tag Product > Untuk mimilih Tag yang diinginkan, dan lalu akan tampil Price Badge 

New Update : 
Add new select type badge, Image or Text.
